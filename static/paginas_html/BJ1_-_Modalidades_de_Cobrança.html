BJ1_CODIGO Codigo Codigo Este campo sera alimentado automaticamente com um codigo sequencial que identificara a Forma de Cobranca. Todos os cadastros serao controlados a partir deste codigo nao sera permitida duplicidade. C 3 0 @! ExistChav("BJ1",M->BJ1_CODIGO,1) Sim Vizualiza Virtual 1 1
