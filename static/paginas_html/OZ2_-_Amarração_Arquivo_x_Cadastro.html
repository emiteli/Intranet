OZ2_ALTFIL Alt.Filial Alterar Filial Quando um arquivo é exportado, o sistema grava a informação da filial origem nos registros enviados. Se este campo for preenchido com "S", o sistema altera o código da filial origem pelo código da filial destino no arquivo texto gerado. Este recurso deve ser utilizado para arquivos exclusivos. Exemplo: O Arquivo SE2 gerado pela filial 01 tem gravado no campo "E2_FILIAL" o conteúdo "01", quando exportado para a filial "02", o arquivo destino terá gravado no campo "E2_FILIAL" o conteúdo "02".
