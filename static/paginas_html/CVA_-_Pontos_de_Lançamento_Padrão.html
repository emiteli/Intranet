CVA_CFGLAN Cfg.OnLine ? Configuração OnLine ? Indica se a configuração do lancamento padrão será efetuada de modo On-Line utilizando-se o assistente de configurações de lancamentos On-Line. Atenção! Este campo deve ser ativado apenas para configuração do sistema devendo ser desativado após a configuração ou manutenção dos lançamentos padrões. A opção "Apenas Administrador" poderá ser utilizada para permitir que a tela do assistente de configuração seja chamada durante a execução do lançamento apenas no login do usuário Administrador possibilitando a utilização do assistente a qualquer momento e apenas pelo Administrador. C 1 0 Pertence("123 ") Sim Altera Real 1=Sim;2=Nao;3=Apenas Administrador 1 2
