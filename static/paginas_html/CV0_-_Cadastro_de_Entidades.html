CV0_ECVM Var.Monet. Entid.Variacao Monetaria Indica a Entidade de Variação Monetária. Será nesta Entidade que serão lançados os valores apurados de Variação Monetária durante o processamento da rotina. Se este campo não for preenchido, a rotina de Variação Monetária não considerará esta entidade para efeito de cálculo. C 6 0 @! Ctb050VM() CV0 Sim Altera Real 1 2
CV0_ECRED Red.Var.M. Redutora Var.Monetaria Indica a Entidade de Redução Var.Monetária. Se informada, esta entidade será utilizada como contra-partida do lançamento contábil de Variação Monetária,deixando dessa forma intactos os saldos da entidade na qual se apurou a Variação Monetária.
