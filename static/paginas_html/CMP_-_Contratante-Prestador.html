CMP_FILIAL Filial Filial Deve representar neste campo o código da filial referente ao cadastro. C 2 0 Sim Altera Real 033 1 2
CMP_ID ID Identificador do registro Deve representar neste campo o código de identificação do registro. C 6 0 @! Sim Vizualiza Real 1 2
CMP_VERSAO Id. Ver. Reg Id da versão do registro Campo de uso interno do sistema. Deve representar neste campo a identificação da versão do registro. Será composto por Data + Hora da ação (Inclusão/Alteração/Exclusão). C 14 0 @! Nao Altera Real 078 1 2
CMP_INSCES Nr. Inscr. Numero Inscrição Deve representar neste campo o número de inscrição do contribuinte, conforme indicado no campo tpInscricao. Se o campo tpInscricao for igual a [1], deve ser um número de CNPJ válido. Se o campo tpInscricao for igual a [2], deve ser um CPF válido. C 14 0 @! Nao Altera Real 1 2
CMP_TPINSC Tp. Inscr. Tipo Inscrição Deve representar neste campo o código correspondente ao tipo de inscrição, conforme tabela 5. Deve ser igual a [1] (CNPJ) ou [2] (CPF). C 1 0 @! Pertence(" 12") Sim Altera Real 1=CNPJ;2=CPF 1 2
CMP_INSCCO Nr. Inscr. Número Inscrição Deve representar neste campo o número de Inscrição (CNPJ/CPF) do Contratante. Deve ser um número de CNPJ ou CPF válido, conforme definido em no campo tpInscContratante. C 14 0 @! XFUNVldPJF(,2,.F.) Sim Altera Real 1 1
CMP_VLBRUT Val. Bruto Valor Bruto Deve representar neste campo o valor bruto da(s) nota(s) fiscal(is) Deve corresponder a soma do campo vlrBruto dos registros vinculados. N 16 2 @E 9,999,999,999,999.99 Sim Altera Real 1 2
CMP_VLSERV Val. Serviço Valor Serviços Deve representar neste campo a soma do valor dos serviços das notas fiscais. Deve corresponder a soma do campo vlrServicos dos registros vinculados. N 16 2 @E 9,999,999,999,999.99 Sim Altera Real 1 2
CMP_BCRETE Bc. Retenção Base de Calculo Retenção Deve representar neste campo soma da base de cálculo da retenção das notas fiscais. Deve corresponder a soma do campo vlrBaseRetencao dos registros vinculados. N 16 2 @E 9,999,999,999,999.99 Sim Altera Real 1 2
CMP_VLRETE Val. Ret. Valor Retenção Deve representar neste campo a soma do valor da retenção das notas fiscais de serviço emitidas para o contratante. Deve corresponder a soma do campo {vlrRetencao} dos registros vinculados. N 16 2 @E 9,999,999,999,999.99 Sim Altera Real 1 2
CMP_VLADIC Val. Adic. Valor Adicional Deve representar neste campo a soma do valor do adicional de retenção das notas fiscais. Deve corresponder a soma do campo vlrAdicional dos registros vinculados. N 16 2 @E 9,999,999,999,999.99 Sim Altera Real 1 2
CMP_VLNRET Val. Não Ret Valor Não Retido Deve representar neste campo o valor da retenção que deixou de ser efetuada pelo contratante ou que foi depositada em juízo em decorrência da decisão judicial. Deve corresponder a soma do campo vlrNaoRetido dos registros vinculados as notas fiscais. N 16 2 @E 9,999,999,999,999.99 Sim Altera Real 1 2
CMP_BCCOOP BC. Coop. Base de Calc. Cooperativa Deve representar neste campo o valor total da base de cálculo da(s) nota(s) fiscal(is) de serviços. Deve corresponder a soma do campo vlrBaseCoop dos registros vinculados ao registro atual. N 16 2 @E 9,999,999,999,999.99 Sim Altera Real 1 2
