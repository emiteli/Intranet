PU_TIPOBRW Tipo Tipo do Grafico C 30 0 @! Sim Vizualiza Virtual PN320ITIPO() 1 2
PU_ALIAS Arquivo Arquivo Fonte de Dados Neste campo deve ser informado o Alias (Apelido) do Arquivo Fonte dos Dados do Gráfico. C 3 0 @! PnVSPUAlias() Sim Altera Virtual #GrafRetOpcBox(02) #PU_ALIAS <>' ' 1 1
PU_CAMPO Campo Campo do Arquivo Neste campo deve ser informado o Nome do Campo do Alias a ser utilizado para obter os dados do Gráfico. Por exemplo, para obter um gráfico dos códigos de apontamento, informamos o campo PC_PD. C 10 0 @! PnVSPUCampo() Nao Altera Virtual #GrafRetOpcBox(03) #PU_CAMPO <>' ' 1 1
PU_CPOBRW Campo Campo do Arquivo C 30 0 @! Sim Vizualiza Virtual PN320ICAMPO() 1 2
PU_CHAVE Ordem Ordem para Totalizacao Neste campo deve ser informado a Ordem de Agrupamento / Classificação. Por exemplo, a ordem "FILIAL+PD" determinará que sejam geradas séries totalizadas por filial e código de apontamento. C 160 0 @!S70 PnvSPUChave() Sim Altera Virtual 1 1
