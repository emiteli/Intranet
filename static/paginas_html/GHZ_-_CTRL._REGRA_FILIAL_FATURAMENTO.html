GHZ_PRIORI Prioridade Prioridade Informe a prioridadde a ser considerada para esta regra. Neste campo, o sistema informa automaticamente o conteúdo 9999, porém caso exista uma regra com mais informações do que esta ela será priorizada. Caso esta regra possua menos informação e seja a que será priorizada, informe 0001. (A prioridade para regras que possuem este campo como 9999, é feita com base na quantidade de informações cadastradas na rotina Filial de Faturamento). C 4 0 9999 Sim Altera Real 1 2
