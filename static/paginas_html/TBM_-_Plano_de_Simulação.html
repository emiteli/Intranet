TBM_CODPLA Plano Codigo do Plano de Simul. C 6 0 @! Sg240Seq() Sim Vizualiza Real 1 1
TBM_DATPLA Data Plano Data do Plano de Simul. Informe a data de inclusão do plano de simulação. D 8 0 99/99/9999 IF(INCLUI,DDATABASE,TBM->TBM_DATPLA) Sim Altera Real 1 1
TBM_DESPLA Descricao Desc. do Plano de Simul. Informe a descrição do plano de simulação C 80 0 @! Sim Altera Real 1 1
TBM_DATINI Data Inicial Data de Inicio do Plano Informe a data inicial de vigência do plano de simulação. D 8 0 VALDATA(M->TBM_DATINI,M->TBM_DATFIM,"DATAMAIOR") IF(INCLUI,DDATABASE,TBM->TBM_DATINI) Nao Altera Real 1 2
TBM_DATFIM Data Final Data de Finalizacao Informe a data de finalização do plano de simulação. D 8 0 VALDATA(M->TBM_DATINI,M->TBM_DATFIM,"DATAMENOR") IF(INCLUI,DDATABASE,TBM->TBM_DATFIM) Nao Altera Real 1 2
TBM_PLAINI Plano Inic. Codigo Inicial do Plano Informe a partir de qual Plano Emergencial deseja gerar o plano de simulação. Pressione a tecla [F3] para selecionar um plano emergencial ou deixe o campo em branco para considerar todos. C 6 0 @! TBB S Nao Altera Real 1 2
TBM_DESPIN Descricao Descricao do Plano Inic. C 80 0 @! TBB->(VDISP(M->TBM_PLAINI,"TBB->TBB_DESPLA")) Nao Vizualiza Virtual 1 2
TBM_PLAFIM Plano Final Codigo do Plano Final Informe até qual Plano Emergencial deseja gerar o plano de simulação. Pressione a tecla [F3] para selecionar um plano emergencial ou digite ZZZZZZ neste campo para considerar todos. C 6 0 @! TBB S Nao Altera Real 1 1
TBM_DESPFI Descricao Descricao do Plano Final C 80 0 @! TBB->(VDISP(M->TBM_PLAFIM,"TBB->TBB_DESPLA")) Nao Vizualiza Virtual 1 2
TBM_SITUAC Situacao Situacao do Plano C 1 0 @! IF(INCLUI,"3",TBM->TBM_SITUAC) Sim Vizualiza Virtual 1=Cancelado;2=Liberado;3=Pendente 1 2
TBM_TERMIN Terminado Plano de Simul. Terminado C 1 0 @! IF(INCLUI,"2",TBM->TBM_TERMIN) Sim Vizualiza Virtual 1=Sim;2=Nao 1 2
