PZ_FILIAL Filial Filial Neste campo deve ser digitado: Código que identifica a filial da empresa usuária do sistema. C 2 0 XM0 Nao Altera Virtual 033 1 2
PZ_NUMERO Numero Numero da Visita Número Contador de Visitas. Ele identifica a qual visita as marcações de acesso pertencem. C 9 0 @! M->PY_NUMERO Sim Vizualiza Virtual 1 1
PZ_VISITA Visitante Numero do Visitante Informe o Código do Visitante que será utilizado como identificação do mesmo. C 6 0 @! NaoVazio() .And. EXISTCHAV("XRC") .And. Val(M->PZ_VISITA) > 0 .And. FreeForUse("SPZ",M->PZ_VISITA) Sim Altera Virtual 1 1
PZ_CRACHA Nr. Cracha Nro. do crachá Informe o número do crachá do Visitante que será utilizado pelo Sistema para identificação das informações geradas pelo relógio/catraca durante a visita. C 10 0 @! NaoVazio() .And. Val(M->PZ_CRACHA) > 0 Sim Altera Virtual 1 2
PZ_DATA Data Data da marcação Informe a data em que foi efetuada a marcação. D 8 0 NAOVAZIO() DDATABASE Sim Altera Virtual 1 1
PZ_HORA Horario Horario da marcação Informe as Horas em que foi efetuada a marcação. N 5 2 @E 99.99 VLDHORA(VAL(M->PZ_HORA)) Sim Altera Virtual 1 2
PZ_RELOGIO Nro. Relógio Número do Relógio Informe o Número do relógio. C 3 0 999 EXISTCPO("SP0",M->PZ_RELOGIO) SP0 Sim Altera Virtual 1 2
PZ_TPMARCA Tp Marcacao Tipo da Marcacao Corresponde as Marcações do Visitante. Ex.: E - Entrada S - Saída M - Marcação de Acesso C 1 0 @! Pertence("123") Sim Altera Virtual 1=Entrada;2=Saida;3=Acesso 1 2
PZ_FLAG Flag Flag da Geracao Corresponde ao "Flag" de Geração das Marcaçöes. C 1 0 ! "I" Sim Vizualiza Virtual 1 2
PZ_FUNCAO Funcäo Funcäo do Relogio Informe a Função do relógio. C 2 0 ! Sim Altera Virtual 1 2
PZ_GIRO Sentido Giro Sentido Giro do Relogio Informe o Sentido do giro relógio. C 1 0 ! Sim Altera Virtual 1 2
