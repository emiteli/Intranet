CTM_DESCM Desc Mascara Descricao da Mascara Identificador da Máscara de Entidade. Campo Informativo. C 40 0 Sim Altera Virtual 1 2
CTM_SEGMEN Segmento Segmento da Conta Segmento da Mascara contábil. C 2 0 !! Sim Vizualiza Virtual 1 1
CTM_DIGITO Dig. do Seg Digitos do Segmento Quantidade de dígitos de um Segmento da máscara. Deverá ser preenchido sempre com dois caracteres. Ex: Para que o sistema imprima uma entidade contábil desta forma: 1.1.11 A máscara deverá ser cadastrada assim: Seg 01 -> "01" digito Seg 02 -> "01" digito Seg 03 -> "02" digitos OBS: NUNCA cadastre o código das entidades contábeis (Conta, Centro de Custo, Item Contábil e Classe de Valor) com os pontos (ou outros caracteres de separação)!! É função da máscara contábil inserir tais separadores no código da entidade contábil. C 2 0 !! Ct130Dig() Sim Altera Virtual 1 1
CTM_DESC Desc Segmen Descricao do Segmento Descrição do Segmento da Máscara. Campo Informativo. C 40 0 @! Sim Altera Virtual 1 2
CTM_SEPARA Caract Separ Caracter de Separacao Caracter de separação entre os dígitos da máscara da entidade contábil. Ex: 1.1/11 Seg 01 -> "." Seg 02 -> "/" Seg 03 -> não informar! C 1 0 ! Ct130Sep() Sim Altera Virtual 1 1
CTM_IDENT Identificad. Identificador Identificador do tipo de Máscara. Poderá ser: - Conta - Centro de Custo - Item Contábil - Classe de Valor - Outros N 1 0 Nao Altera Virtual 1 2
