RBR_VLREF Valor Refer. Valor de Referencia Neste campo deve ser informado o valor de referência para cálculo dos coeficientes ou valores das faixas da tabela salarial. N 12 2 @E 999,999,999.99 Sim Altera Real 1 2
RBR_USAPTO Usa Pontos Utiliza Pontos Min. e Max Neste campo deve ser informada a utilização de pontos mínimos e máximos na tabela salarial: 1- Sim ou 2- Não N 1 0 PERTENCE("12") Nao Altera Real 1=Sim;2=Nao 1 2
RBR_APLIC Aplicada Aplicada Neste campo deve ser informado se a tabela salarial é utilizada ou não para o reajuste salarial de funcionários: 1- Sim ou 2- Não. C 1 0 9 PERTENCE("12") "2" Nao Altera Real 1=Sim;2=Nao 1 2
RBR_TIPOVL Tipo Valor Tipo Vl 1-Exato 2-Interv. Neste campo deve ser informado o tipo da Tabela Salarial: 1- Por valores exatos ou 2- Por valores com Intervalos. N 1 0 Nao Altera Real 1 2
RBR_SIMBOL Simbolo Simbolo informe o símbolo para esta tabela salarial. Será utilizado em relatórios. C 10 0 @! Nao Altera Real 1 2
RBR_TABINI Tab.Ini.Carr Tabela Inicio da Carreira Informe o código da tabela salarial a ser considerada como "Início de Carreira". Essa informação é importante para cálculos de folha de pagamento que utilizam a tabela de início de carreira como base de cálculo. C 3 0 @! Sim Altera Virtual 1 2
