FTI_CODIGO Referencia Numero de Referencia Código que identificará o grupo de despesas, ou o chamado número de referência do lote de despesas. O Sistema atribui o último número em aberto quando o usuário escolhe abrir novo grupo ou usa o grupo em que o cursor está posicionado no browse quando o usuário escolhe apontar a despesa no grupo atual. C 6 0 GETSX8NUM("FTI","FTI_CODIGO") Sim Vizualiza Real 1 2
FTI_STATUS Status Status do Grupo de Desp. Flag que identifica a situação do grupo de despesas, pode ser: 1 - Aberto, que significa que o grupo ainda vai receber apontamento de despesas; 2 - Encerrado, que significa que o grupo não irá mais receber apontamento de despesas; 3 - Liberado para faturamento; 4 - Parcialmente Faturado ou 5 - Faturado. C 1 0 @! "1" Sim Altera Real 1=Aberto;2=Encerrado;3=Liberado para Faturamento;4=Parcialmente Faturado;5=Faturado 1 2
FTI_DTABER Abertura Data de Abertura Data da abertura do grupo de despesas, o sistema assume a data do sistema. D 8 0 Sim Vizualiza Real 1 2
FTI_CODUSU Usuario Usuario Código do usuário que está apontando as despesas. C 6 0 @! Sim Vizualiza Real 1 2
FTI_DTENCE Encerramento Data de Encerramento Data do encerramento do Grupo de Despesas, o Sistema assume a Data do sistema. D 8 0 Sim Vizualiza Real 1 2
FTI_ULTDAT Dt.Ult.Apont Data ultimo apontamento Data do último apontamento de despesas para que seja repetido no próximo apontamento de modo a agilizar a digitação. D 8 0 @D Nao Vizualiza Real 1 2
FTI_ROTCHV Rot. Chave Rotina Chave Informa qual a rotina geradora do grupo de despesa, ou seja, se é despesa digitada para o contas a pagar ou se é proveniente do caixinha. C 1 0 @! "1" Nao Vizualiza Real 1=Contas a Pagar;2=Caixinha 1 2
