DTA_ORIGEM Orig. Carreg Origem do Carregamento Informe o tipo de região a ser considerado para preenchimento das tags do Manifesto Eletrônico que se referem ao município do carregamento. Caso este campo não esteja preenchido será considerado o município da filial responsável pela geração do Manifesto eletrônico e, neste caso, as informações serão obtidas do Cadastro de Empresas/Filiais. 2-Cliente Remetente: Será considerado o município onde se localiza o Cliente Remetente relacionado do documento de transporte. 3-Local de Coleta: Será considerado o município da coleta. Esta opção só poderá ser utilizada para os documentos de transporte que tenham sido gerados a partir de uma nota fiscal que tenha sido vinculada a uma Solicitação de Coleta.
