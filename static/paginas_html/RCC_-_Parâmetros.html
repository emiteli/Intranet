RCC_FILIAL Filial Filial Informe o código da filial. C 2 0 XM0RCB Nao Altera Virtual 033 1 1
RCC_CODIGO Codigo Codigo do Parametro Informe o conteúdo da identificação do item da tabela. Por exemplo, se a tabela em alteração for a tabela de países, esse campo corresponde ao código de identificação do país. C 4 0 @! EXISTCPO("RCB") Nao Vizualiza Real #RCC_CODIGO<>' ' 1 2
RCC_FIL Filial Filial da Tabela Inserir a Filial da tabela. Se filial da tabela for em branco, a tabela será considerada para todas as filiais. Caso contrario será válida somente para a filial escolhida. C 2 0 XM0 Nao Altera Virtual 033 1 2
RCC_CHAVE Mes / Ano Mes / Ano Inserir o mês e ano válidos que deverá ser considerada a tabela. Caso esteja em branco a tabela cadastrada, será válida para qualquer período. Ex.: Mês/Ano : 022002 C 6 0 999999 IIF(EMPTY(M->RCC_CHAVE),.T.,(ENTRE( "01","12",Substr(M->RCC_CHAVE,1,2) ) .AND. LEN(ALLTRIM(M->RCC_CHAVE))==6)) Nao Altera Virtual 1 2
RCC_SEQUEN Sequencia Sequencia Sequencia. C 3 0 @! Nao Vizualiza Real 1 2
RCC_CONTEU Conteudo Conteudo da tabela Informe o conteúdo. C 250 0 @! Nao Altera Virtual 1 2
