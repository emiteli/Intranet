CTL_LP Lcto Padrao Lancamento Padrao Indica o Código de Lançamento Padrão. Será a partir do Lançamento-Padrão que o rastreamento da Origem do lançamento contábil será efetuado. O rastreamento de uma operação só será possível se o lançamento-padrão foi: 1. Cadastrado corretamente 2. O lançamento contábil tenha sido efetuado 3. O relacionamento também tenha sido cadastrado corretamente (utilizando o mesmo código do lançamento padrão). Antes que qualquer lançamento contábil de integração seja executado, este cadastro deverá ser configurado. Caso contrário, esses lançamentos gerados não terão as chaves de rastreabilidade gravadas. C 3 0 @! ExistCpo("CT5",,1) .And. ExistChav("CTL") .And. FreeForUse("CTL",m->ctl_lp) CT5 Sim Altera Virtual 1 1
CTL_KEY Chave Busca Chave de Busca Esta é normalmente a chave primária do arquivo a ser rastreado. Nada impede que o usuário utilize outra, mas deverá ser garantida a unicidade da informação, isto é, a chave deverá garantir que apenas um registro seja encontrado. C 200 0 @! Sim Altera Virtual 1 1
CTL_ORDER Ordem Busca Ordem de Busca Indica qual é a ordem da Chave de Busca cadastrada anteriormente dentro do arquivo de índices. Especifique a ordem correta, pois o sistema irá selecionar a Ordem e efetuar a busca da Origem do Lançamento pela Chave Busca. C 2 0 @! Sim Altera Virtual 1 1
CTL_DESC Descricao Descricao do Relacionam Esta é a descrição da utilização deste Relacionamento. C 40 0 @! Sim Altera Virtual 1 1
CTL_EXECUT Rot Executar Rotina a ser Executada Indica a rotina a ser executada, no momento do Rastreamento da Origem do Lançamento. Esta deverá ser a rotina de visualização do registro. Caso não seja preenchido, o SIGACTB irá executar as suas rotinas próprias. C 40 0 @! Nao Altera Virtual 1 2
CTL_ALIAS Alias Arq Alias da Tabela Indica o Alias do arquivo onde está o registro a ser rastreado. Este campo anexado aos “Chave Busca” e “Ordem Busca” compõem a chave de procura do registro origem do lançamento. Portanto estes 3 campos deverão ser correspondentes à um Índice de tabela já existente no Advanced Protheus. C 3 0 @! Sim Altera Virtual 1 1
